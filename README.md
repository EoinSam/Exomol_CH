# Exomol_CH
Supplementary information for the Molecular line list of the methylidyne radical (CH), including the Duo input and Pseudo-Marvel File
The CH_int_68 has the most up to date spectroscopic model and was the one used to create the spectroscopic graphs in the Dissertation. A states and compressed trans file for CH_int_68 have been included at 1000K for the bound states and transitions.

The CH_71.out file is included to provide the most recent parameters and Marvel fitting for the X state.
